System Requirements:
--------------------

NodeJs v8 or higher
npm
nvm
curl
git


Installing Node
---------------
cd /home/
sudo apt-get update
sudo apt install curl
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo apt-get install -y build-essential


Installing Node using NVM (Recommended)
-------------------------
sudo apt install curl
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.0/install.sh | bash
nvm install node
nvm use node

Installing V-Track Client Agentls
------------------------------
cd /home/
sudo apt-get install git
sudo git clone https://github.com/superdevdave/vtracklinux/
run the command which node to verify location of nodejs
navigate to vtracklinux folder
run vi vtrack.service and modify path of node js on the ExecStart parameter

cp /home/davidm/vtacklinux/vtrack.service /etc/systemd/

sudo systemctl enable /etc/systemd/vtrack.service

cd /home/davdm/vtracklinux

sudo chown -R <yourusername> machine.jaon
